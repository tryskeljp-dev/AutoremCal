Connectez Google Agenda

Affichez tous vos calendriers dans une vue multi-personnes style AutoremCal. Vos données restent locales.

Première utilisation ?

1
Allez sur console.cloud.google.com → Nouveau projet
2
APIs → Activez Google Calendar API
3
Identifiants → OAuth 2.0 → Application Web → Ajoutez file:// dans les origines autorisées
4
Copiez votre Client ID et collez-le dans le code (GOOGLE_CLIENT_ID)
